There are five subwindows in gitk graphic interface. 
1. upper-left:
This subwindow shows the series of commits to this repository, using points to represent commit, a simple sentence to describe the commit. The most recent one is on the top. 

2. upper-middle:
It shows the author information corresponding to the commit in upper-left subwindow of the same row.

3. upper-right:
It shows the change time of commit, also corresponding to the commit in same row.

4. lower-left:
This subwindow shows specific file changes, usually in diff format. Meanwhile, there are Diff, Old version and New version choices which you choose to compare the changes. 

5. lower-right:
This subwindow shows files impacted by selected commit. You can also see patch and tree by different choices.
